# MkDocsの利用

## 動作確認

まずシンプルな文書を配置します。

```bash
cp -rf reference/01.mkdocs/docs/index.md docs/
```

MkDocsを起動します。

```bash
uv run mkdocs serve --livereload
```

リンクをクリックして表示を確認する。

URL: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## タスクランナーを利用する

ここではpnpmのscriptsをniから利用します。

[packages.json](../package.json)のscriptsを確認してください。

つぎのように利用できます。

```bash
nr mkdocs
```

## 便利な拡張の紹介

### 検索機能の追加

ルートのmkdocs.ymlを開き、下記を追加します。

```yml
plugins:
  - search
```


### Admonitionsの追加

ルートのmkdocs.ymlを開き、下記を追加します。

```yml
markdown_extensions:
  - admonition
  - pymdownx.details
  - pymdownx.superfences
```

index.mdに下記を追加してブラウザをリロードしてください。

```markdown
## Admonition

!!! note

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod
    nulla. Curabitur feugiat, tortor non consequat finibus, justo purus auctor
    massa, nec semper lorem quam in massa.

!!! danger

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla et euismod nulla.
    Curabitur feugiat, tortor non consequat finibus, justo purus auctor massa, 
    nec semper lorem quam in massa.
```
