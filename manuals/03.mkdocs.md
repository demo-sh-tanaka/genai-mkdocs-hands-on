# MkDocsの利用

```bash
cp -rf reference/01.mkdocs/index.md docs/
```


1. docs/index.mdの作成
2. docs/stylesheets/site.cssの作成
3. docs/.pagesの作成
4. mkdocs.ymlの作成

## 5. MkDocsの実行

```bash
uv run mkdocs serve --livereload
```
