# Marpによるスライドの作成

Marpを利用したスライドを作成します。

まず下記を実行して必要なコンテンツをコピーしてください。

```bash
cp -rf reference/08.marp/. docs/
```

.vscode/settings.jsonにmarpのテーマ設定を追加します。

```json
{
  "markdown.marp.themes": [
    "./docs/スライド/theme/marp.css"
  ],
```

Marpを起動する。

MkDocsとポートを共有しているので、MkDocsは先に停止させてください。

```bash
# 直接実行
marp -s docs/スライド

# niによる実行
nr marp
```

## MkDocsにPDFを配置する

コンソールからPDFをビルドして配置します。

```bash
# 直接実行
marp --input-dir docs/スライド --output docs/スライド --pdf --allow-local-files

# niによる実行
nr marp:build
```

Marpを停止した後、MkDocsを起動します。

```bash
nr mkdocs
```
